Consulta Previsão do Tempo

App para consulta de Previsão do Tempo

Olá, este é um projeto cuja finalidade é realizar rápidas consultas de previsão de tempo de várias localidades espalhadas pelo mundo.

O principal objetivo com este trabalho era realizar minha primeira integração com API em um projeto.

Desta forma foi utilizado como fonte principal para realização deste trabalho a API de dados da Open Weather Maps.

Uso: Para utilizar basta prencher a barrar de pesquisa com o nome da cidade que deseja informações.

Há casos de cidades com mesmo nome em diferentes partes do mundo, este problema é facilmente resolvido no momento que o usuário realizar a busca mais precisa digitando o nome da cidade que deseja buscar + a sigla do seu país.
Ao exemlo: Salvador, BR -- Manaus, BR -- São Paulo, BR.
